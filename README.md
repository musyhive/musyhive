# MusyHive

MusyHive is a social media app built with Node.js and Express.

## Getting Started

1. **Clone the Repository:**
```bash
   git clone https://github.com/ongoingend/musyhive.git
```
   
## Install Dependencies

```bash
cd musyhive
npm install
npm install nodemon
```

## Run the Application

```bash
npx nodemon
```
